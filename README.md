## 🛠️ Stack Technique
- React + TypeScript (Vite)
- Tailwind CSS (Thème Luxury Dark & Gold)
- Lucide React (Icons)

## ✨ Features implémentées
- **Landing Page Immersive** : Design haut de gamme avec responsive mobile-first.
- **Catalogue de Destinations** : Cartes interactives pour Paris 1889, Florence 1504 et le Crétacé.
- **Agent Conversationnel (Chronos)** : Chatbot IA simulant la conciergerie de luxe et l'ajustement des tarifs de saut.
- **Quiz de Personnalisation (Optionnel 3.2)** : Questionnaire intelligent en 4 étapes recommandant l'époque idéale.
